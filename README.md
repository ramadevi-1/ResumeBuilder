
## Dependencies

- **Font Awesome 6.4.0** - Icons
- **jsPDF 2.5.1** - PDF generation

## How to Use

1. **Fill in Personal Information**
   - Enter your name, job title, contact details
   - Write a professional summary

2. **Add Work Experience**
   - Click "Add Experience" to add new entries
   - Fill in job title, company, duration, and description
   - Click the "×" button to remove entries

3. **Add Education**
   - Click "Add Education" to add new entries
   - Fill in degree, institution, duration, and description
   - Click the "×" button to remove entries

4. **Add Skills**
   - Enter skills separated by commas

5. **Preview & Save**
   - Watch the live preview update as you type
   - Click "Save Resume" to save to browser storage
   - Click "Download PDF" to export as PDF

## Browser Compatibility

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+

## Technical Details

- **No Backend Required**: All data stored in localStorage
- **Responsive Design**: Works on desktop, tablet, and mobile
- **PDF Generation**: Client-side PDF creation using jsPDF
